# Quotex Signal Bot

This bot generates 1-minute binary options signals (CALL / PUT) for selected markets and sends alerts to your Telegram.

## Setup Instructions

1. Install Python 3.10+
2. Install libraries:
   pip install -r requirements.txt

3. Open quotex_signal_bot.py
   - Replace YOUR_TELEGRAM_BOT_TOKEN
   - Replace YOUR_CHAT_ID

4. Run the bot:
   python quotex_signal_bot.py

You will receive live signals in your terminal and Telegram every minute.
